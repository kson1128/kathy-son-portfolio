iBook Shop
IBook Shop is a web application that is built with Next.js. This framework has improved initial page load times through Next.js server-side rendering (SSR). By implementing Redux, it facilitates data flow across components, providing a scalable and predictable approach to managing the application's global state. Moreover, it utilizes Express to establish a reliable and efficient server-side infrastructure capable of handling API requests and authentication seamlessly.
